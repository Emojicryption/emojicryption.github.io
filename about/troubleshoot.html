Troubleshoot

Solutions to Problems that might happen
Sorry, you are OFFLINE

Open your Settings app > "Wireless & networks" or "Connections". Depending on your device, these options may be different. Check the connection status. Turn Wi-Fi/mobile data on, and check to see if there's a difference.

Restart your device. It might sound simple, but sometimes that's all it takes to fix a bad connection.

YOU DIDN'T WRITE
Get back and Try Again

This error can show up, while doing an encryption. Make sure you write some text, before you press the Encrypt button.

Now Press BACK, write something (Don't keep the text-box empty) and try again.

NO EMOJIS ???

This error can show up, while doing an decryption. Make sure you type or paste some emojis, before you press the decrypt button.

Now Press BACK, type or paste your emojis (Don't keep the text-box empty) and try again.

Seeing 'undefined'
in encrypted text

This error can show up, while doing an encryption. Currently, you can encrypt only plain text (along with numbers and alphanumerals. Make sure there isn't any emoji with the plain text.

Try to do it again, without giving an emoji in the plain text. It should work now.

WRONG EMOJIS
There's something wrong

This error can show up, while doing an decryption. Make sure you've written the correct emojis, in the exact order you got it. Best is to COPY those recieved emojis and PASTE them while decryting.(So that you don't get this error)

Now Press BACK, type or paste your emojis (Don't keep the text-box empty) and try again.
